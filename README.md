# XServer Mail Blacklist

MailKit extension and macOS companion app for managing XServer email blacklists directly from Apple Mail.

## Features

- Add sender addresses to XServer blacklists directly from Apple Mail
- Support multiple XServer accounts and API credentials
- Manage blacklist entries from a macOS companion app
- Integrate with XServer API, CLI, and MCP

## Requirements

- macOS 15+
- Apple Mail
- XServer Mail Account
- XServer API Key
